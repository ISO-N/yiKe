YiKe / Material 3 Mobile Prototype

忆刻手机 App UI/UX 原型

这一版围绕“打开就能开始复习”来设计，采用 Material 3 常见的色彩语义、卡片层级、 底部导航和 FAB 语言，同时保留离线学习工具应有的克制感。

主色 / Primary

用于开始复习、关键确认按钮和流程进度，保持任务导向感。

容器色 / Primary Container

承接今日任务、轻状态提示与顶部概览，减少大面积纯色压迫。

强调色 / Tertiary

用于 FAB、备份风险和轻提醒，让高关注动作更容易被看到。

表面层 / Surface

大面积背景保持浅色高对比，更适合长时间阅读题面和答案。

信息架构

首页、卡组、设置三段式底部导航承接日常入口；复习流作为独立沉浸式页面； 卡组详情和问题编辑延续 Material 列表与表单逻辑，减少学习成本。

首页

任务总览、开始复习、近期卡组入口

复习页

问题聚焦、显示答案、四档评分

卡组列表

主题聚合、新建卡组、今日到期信息

卡片列表

章节级内容管理、进入问题编辑

问题编辑

卡片信息 + 多问题编辑，贴近移动端表单习惯

设置

提醒、版本信息、备份恢复入口

备份与恢复

高风险操作清晰分层，减少误触
