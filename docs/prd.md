# 忆刻（YiKe）— 产品需求文档

> 统一沉淀产品需求与验收口径 | 最后更新：2026-03-05

---

## 1. 项目概述

### 1.1 产品定位

忆刻是一款基于**艾宾浩斯遗忘曲线**的学习任务规划应用：用户录入学习内容，系统自动生成复习任务，通过通知/小组件/桌面端持续提醒。

### 1.2 核心价值

| 原则 | 说明 |
|------|------|
| **零门槛** | 输入即规划，不强制用户理解记忆曲线理论 |
| **可执行** | 明确"复习什么、什么时候复习、今天做多少" |
| **可追踪** | 完成/跳过有状态闭环，统计与进度实时反馈 |
| **跨设备可用** | 移动端/桌面端体验一致，局域网内数据同步 |

### 1.3 目标用户

| 用户画像 | 典型场景 |
|----------|----------|
| 学生 | 备考、课程复习、背诵记忆 |
| 职场人 | 考证、技术面试、知识点复盘 |
| 终身学习者 | 语言学习、读书笔记、长期复习 |

### 1.4 核心概念

| 概念 | 定义 |
|------|------|
| **学习内容** | 用户录入的一条知识点（标题/描述/子任务/标签/学习日期） |
| **复习任务** | 系统为学习内容生成的待复习项（轮次 1-10、计划日期、状态） |
| **状态** | 待复习（pending）/ 已完成（done）/ 已跳过（skipped） |
| **复习轮次** | 默认 1-10 轮，可在设置中启用/禁用轮次并调整间隔 |

## 2. 范围定义

### 2.1 在范围

- 离线优先：核心功能无网络可用
- 跨平台：Android / iOS / Windows
- 局域网同步：设备发现、配对、双向同步
- Debug 工具：仅 Debug 模式可见

### 2.2 暂不在范围

- 账号体系、云端登录、多端云同步
- 社区/公开内容/排行榜等社交属性
- AI 自动生成学习内容

## 3. 功能需求

### 3.1 录入与内容管理

**目标**：低成本创建高质量学习内容，明确复习路径。

- **录入字段**：标题（必填 ≤50字）、描述（选填）、子任务（选填）、标签（选填）、学习日期（默认今日）
- **连续录入**：保存成功后可快速继续录入
- **批量导入**：TXT/CSV/Markdown → 预览/编辑 → 一次性入库
- **导入引导**：预览页提供"格式说明 + 下载模板"
- **模板管理**：支持占位符（如日期占位符）
- **语音/OCR**：语音输入与图片识别
- **复习预览**：录入时预览复习节点，支持配置间隔
- **主题关联**：内容归到主题，主题页聚合展示进度

### 3.2 复习任务闭环

**默认复习间隔**（D0 起）：D1/D2/D4/D7/D15/D30/D60/D90/D120/D180

- **自动生成**：按间隔配置生成 1-10 条复习任务
- **完成/跳过**：支持批量操作
- **逾期提示**：明显标识（颜色/标记）
- **可逆操作**：已完成/已跳过支持撤销，明确提示对统计的影响
- **任务详情**：展开查看描述/子任务/标签，提供编辑/调整计划/停用入口

### 3.2.1 全量任务视角

- 首页提供"今日 / 全部"切换（`/home?tab=all`）
- 时间线按发生时间排序（pending 用 scheduledDate，done 用 completedAt）
- all-tab 支持**状态 + 时间**双重筛选
  - 状态：全部 / 待复习 / 已完成 / 已跳过
  - 时间：全部 / 今天前 / 今天后（基于 `scheduledDate`）
  - 时间边界：今天前=`scheduledDate < 今天00:00`；今天后=`scheduledDate >= 明天00:00`
  - 今天任务仅在时间筛选为"全部"时出现
  - "未完成"口径仅指 pending，skipped 保持独立查看
- 游标分页 + 列表虚拟化

### 3.2.2 任务操作增强

- **编辑内容**：修改描述/子任务/标签（影响所有轮次）
- **停用**：软删除，详情页只读
- **调整计划**：仅允许调整 pending 轮次，满足相邻轮次约束
- **轮次管理**：支持增/减轮次（1-10），减少时提示影响

### 3.3 提醒与小组件

- 通知提醒：默认 09:00，支持免打扰时段，权限引导
- 桌面小组件：2×2 显示数量，4×2 显示列表

### 3.4 日历视图

- 月历展示任务分布，点击日期展示当日任务

### 3.5 学习统计

- 连续天数、周/月完成率、分类/标签分布
- 数据口径一致：统计以任务状态与日期为准

### 3.6 数据导出与备份

- **导出**：学习内容/子任务/复习任务/复习记录/设置（默认排除 Mock）
- **格式**：JSON/CSV，兼容旧表头
- **备份恢复**：
  - UTF-8 JSON（`.yikebackup`），支持 SHA-256 校验
  - 默认合并导入，覆盖导入需快照 + 二次确认
  - 支持取消/中断

### 3.7 主题系统

- 模式：跟随系统/浅色/深色
- 深色模式全页面适配
- 首次进入主题页提供认知引导

### 3.8 学习指南与帮助

- 独立全屏页面（`/help`），桌面端 Ctrl+H 打开
- Markdown 渲染，深浅主题适配

### 3.9 Windows 桌面端

- Windows 10+ 支持，键鼠优化
- 窗口/托盘/快捷键/响应式双列布局

### 3.10 局域网数据同步

- **发现**：UDP 广播（端口 19876）
- **传输**：HTTP（端口 19877）
- **认证**：6 位配对码 + 5 分钟有效期
- 双向同步，支持至少 3 台设备

### 3.11 Debug 与体验优化

- **模拟数据生成器**：仅 Debug 可见，可配置数量/范围/模板
- **Mock 数据隔离**：默认不参与同步/导出
- **搜索**：支持标题/描述/备注/子任务，防抖，可取消
- **筛选**：按状态筛选，展示各状态数量
- **进度环**：展示已完成/总数，点击展开详情

## 4. 非功能需求

| 类别 | 需求 |
|------|------|
| 性能 | 冷启动 ≤ 2s，录入到生成计划 ≤ 500ms |
| 存储 | SQLite，10,000+ 任务查询流畅 |
| 兼容性 | Android 8.0+ / iOS 14+ / Windows 10+ |
| 离线 | 核心功能完全离线可用 |
| 安全 | 敏感配置加密，同步需认证 |

---

## 5. 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 通知被限制 | 提醒失效 | 权限引导 + 小组件兜底 |
| 任务堆积 | 体验劣化 | 逾期提示 + 筛选/搜索 + 进度可视化 |
| 局域网发现不稳定 | 同步不可用 | UDP 广播 + 手动 IP 输入 |
| 同步冲突 | 数据丢失 | 明确冲突策略 + 状态可见 |
| Mock 数据污染 | 统计异常 | 标记 + 过滤 + 一键清理 |
