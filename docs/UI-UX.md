# 忆刻（YiKe）— UI/UX 设计

> 统一沉淀设计体系与交互规范 | 最后更新：2026-03-05

---

## 1. 设计目标与原则

### 1.1 设计目标

- **零门槛**：输入即规划，用户不必理解记忆曲线理论
- **专注高效**：关键路径短、反馈明确、减少打断
- **跨平台一致**：移动端与 Windows 桌面端统一信息架构与视觉语言
- **可访问**：对比度、触摸目标、键盘与读屏适配

### 1.2 交互原则

- **重要操作**：导入/删除/清空/同步/生成模拟数据 → 确认/进度/结果反馈
- **空状态**：可行动，告诉用户下一步做什么
- **状态表达一致**：待复习/已完成/已跳过/逾期在所有页面保持统一

## 3. 信息架构与导航

### 3.1 底部导航（4 Tab）

| Tab | 路由 | 说明 |
|-----|------|------|
| 今日 | `/home` | 今日任务与快速操作 |
| 日历 | `/calendar` | 月视图与当天列表 |
| 统计 | `/statistics` | 完成率与分布 |
| 设置 | `/settings` | 提醒/主题/导出/同步/关于 |

- **全量任务**：`/home?tab=all`，旧路由 `/tasks` → redirect
- **帮助页**：`/help`（独立全屏），桌面端 Ctrl+H，`/settings/help` → redirect

### 3.2 Windows 桌面端导航

- 保持同一 IA，允许更高信息密度（双列/侧栏）
- 快捷键映射：新建/刷新/搜索聚焦

### 3.3 IA 精简（规划）

- 目标：4 Tab → 3 Tab（统计整合到日历页）
- 旧路由保持兼容

## 4. 页面规范

### 4.1 首页（今日/全部任务）

- **结构**：搜索 → 今日/全部切换 → 进度 → 筛选 → 任务列表 → FAB（新建）
- **卡片信息**：标题 + 标签 + 第 N 次复习 + 描述/子任务摘要 + 状态控件
- **交互**：
  - 点击：展开详情
  - 长按/右键：上下文菜单
  - 左滑：快捷完成/跳过（仅 pending）
- **空状态**：
  - 冷启动：引导添加内容/创建专题
  - 常规空态：提示已完成，提供添加入口
- **失败回滚**：操作先行，失败不播放移除动效
- **减少动效**：系统开启时降级/禁用

### 4.2 录入页

- **字段**：标题/描述/子任务/标签/学习日期
- **批量导入**：导入 → 预览/编辑 → 确认
- **模板**：选择模板填充，占位符预览
- **复习预览**：显示复习节点，配置入口清晰

### 4.2.1 任务详情 Sheet

- **路由**：`/tasks/detail/:learningItemId`
- **操作**：
  - 编辑描述/子任务/标签
  - 调整 pending 轮次日期
  - 增/减轮次（最大 10，最小 1）
  - 停用（软删除）

### 4.2.2 主题管理页

- **路由**：`/topics`
- 首次进入显示用途说明，支持"不再提示"

### 4.3 日历页

- 月历 + 当日任务列表
- 日期标记状态（颜色/圆点）
- 筛选栏：全部/待复习/已完成/已跳过

### 4.4 统计页

- 关键指标：连续天数、周/月完成率
- 分布：标签/主题饼图

### 4.5 导出页

- 选择对象与格式（JSON/CSV）
- 默认排除 Mock 数据

### 4.5.1 备份与恢复

- 快捷按钮 + 备份历史列表
- 导出：私有目录 → 分享/另存为
- 导入：校验 → 预览 → 选择策略（合并/覆盖）→ 执行

### 4.6 设置页

- 提醒/外观/数据管理/同步/Debug（仅 Debug）

### 4.7 帮助页

- 标题 + 可滚动正文
- Markdown 样式，目录跳转

### 4.8 同步状态

- 状态：未连接/发现中/配对中/已连接/同步中/失败
- 展示：设置页 + 顶部轻提示 + 托盘菜单

### 4.9 Debug 模拟数据生成器

- 仅 Debug 可见
- 可配置数量/范围/模板，一键清理

## 5. 组件规范

### 5.1 按钮

| 类型 | 用途 |
|------|------|
| Primary | 主要行动（保存、导出、同步） |
| Secondary | 次要行动（取消、返回） |
| Ghost | 文字按钮/轻操作 |
| Danger | 删除/清空/断开 |

### 5.2 状态颜色

- 待复习：Primary
- 已完成：Success
- 已跳过：Text Secondary（灰）
- 逾期：Warning

### 5.3 进度环

- 未开始：灰色
- 进行中：Primary（Teal）
- 已完成：Success（Green）
- 文案：`已完成/总数`，点击展开详情

## 6. 动画与反馈

| 场景 | 规范 |
|------|------|
| 页面切换 | 300ms ease-in-out |
| 主题切换 | 背景渐变 300ms，卡片缩放 100ms |
| 关键操作 | 必须有加载态与结束态 |
| 任务完成 | 缩放 + 淡出 ≤300ms |
| 左滑 Dismissible | 保持原生效果 |
| 失败 | 不播放移除动效，提示错误 |
| 无障碍降级 | 减少动态效果时动效时长设为 0 |

---

## 7. 响应式与桌面端

### 7.1 断点

- 移动端：< 600px
- 平板：600px - 1024px
- 桌面端：≥ 1024px（双列/侧栏）

### 7.2 Windows 窗口

- 默认：1200×800，最小：800×600
- 托盘：最小化展示状态

---

## 8. 无障碍

- 对比度：正文 ≥ 4.5:1，大字 ≥ 3:1
- 触摸目标：≥ 44×44，列表项 ≥ 48
- 读屏：语义标签，状态可朗读
- 键盘：可聚焦、可触发

---

## 9. 验收检查清单

- 深浅主题所有页面可读
- 帮助页 Markdown 完整，目录可跳转
- 同步状态展示一致
- 搜索/筛选/进度不遮挡核心任务
- 备份/恢复路径完整
- 任务操作互斥规则正确
- 性能降级开关有效
