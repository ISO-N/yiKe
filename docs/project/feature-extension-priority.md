# 忆刻 (Yike) 功能扩展优先级列表

> 本文档列出可在不改变数据模型的前提下，通过增删查改实现的功能，按优先级排序。

## 优先级说明

| 优先级 | 标记 | 说明 |
|--------|------|------|
| P0 - 紧急 | 🔴 | 高用户价值，实现简单，应优先开发 |
| P1 - 高 | 🟠 | 较高价值，安排在下一迭代 |
| P2 - 中 | 🟡 | 中等价值，后续迭代安排 |
| P3 - 低 | 🟢 | 低优先级，可选开发 |

---

## 🔴 P0 - 紧急（数据完备，实现简单）

### 1. 复习统计页面
- **功能描述**：展示学习进度和效果的数据统计
- **利用字段**：`Question.reviewCount`, `lapseCount`, `lastReviewedAt`, `stageIndex`, `ReviewRecord.rating`, `responseTimeMs`, `reviewedAt`
- **实现方式**：新增 DAO 聚合查询 + 统计页面 UI
- **具体功能**：
  - 连续学习天数统计
  - 评分分布统计（AGAIN/HARD/GOOD/EASY 比例）
  - 遗忘率统计
  - 平均响应时间

### 2. 问题搜索与筛选
- **功能描述**：快速找到需要的问题
- **利用字段**：`Question.prompt`, `answer`, `tags`, `status`
- **实现方式**：新增 DAO LIKE 查询
- **具体功能**：
  - 全文搜索问题/答案
  - 按标签筛选
  - 按状态筛选（进行中/已归档）
  - 按卡组/卡片筛选

### 3. 今日复习预览
- **功能描述**：开始复习前了解今天要复习的内容概览
- **利用字段**：`Question.dueAt`, `prompt`, `Card.title`, `Deck.name`
- **实现方式**：现有数据聚合展示
- **具体功能**：
  - 今日待复习问题列表预览
  - 按卡组分类显示今日任务
  - 预计复习时长估算（基于平均 responseTimeMs）

### 4. 知识熟练度标签
- **功能描述**：基于复习数据自动为问题打上熟练度标签
- **利用字段**：`Question.reviewCount`, `stageIndex`, `lapseCount`
- **实现方式**：基于字段计算，无需存储
- **具体功能**：
  - 自动标记"新问题"、"学习中"、"熟悉"、"已掌握"
  - 按熟练度筛选问题
  - 熟练度可视化进度条

---

## 🟠 P1 - 高优先级

### 5. 错题本 / 难点卡片
- **功能描述**：专门收集"完全不会"或频繁遗忘的问题
- **利用字段**：`Question.lapseCount`, `ReviewRecord.rating`
- **实现方式**：按遗忘次数排序查询
- **具体功能**：
  - 标记高遗忘次数的问题为"难点"
  - 一键查看所有 AGAIN 评分的问题
  - 错题专项复习入口

### 6. 标签筛选与管理
- **功能描述**：利用已有的标签字段实现筛选功能
- **利用字段**：`Question.tagsJson`
- **实现方式**：新增 Repository 查询方法
- **具体功能**：
  - 按标签筛选问题
  - 标签管理界面
  - 跨卡组标签视图

### 7. 批量操作
- **功能描述**：提升内容管理效率
- **利用字段**：`Question.status`, `deckId`, `cardId`, `tags`
- **实现方式**：批量 upsert/delete
- **具体功能**：
  - 批量归档/恢复问题
  - 批量移动问题到其他卡片
  - 批量添加/移除标签
  - 批量删除

### 8. 复习历史回顾
- **功能描述**：查看问题的历史复习记录
- **利用字段**：`ReviewRecord` 完整记录
- **实现方式**：新增详情页展示
- **具体功能**：
  - 问题详情页显示复习历史时间线
  - 查看每次复习的评分和阶段变化
  - 复习效果回顾

---

## 🟡 P2 - 中优先级

### 9. CSV 导出
- **功能描述**：扩展备份导出格式
- **利用字段**：全部现有字段
- **实现方式**：新增 CSV 序列化器，复用备份架构
- **具体功能**：
  - 全量 CSV 导出
  - 按卡组单独导出
  - 选择性导出（仅导出未归档内容）

### 10. 响应时间分析
- **功能描述**：分析用户回答速度
- **利用字段**：`ReviewRecord.responseTimeMs`
- **实现方式**：统计平均值/分布
- **具体功能**：
  - 平均响应时间展示
  - 响应时间趋势图
  - 熟练度参考指标

### 11. 学习时段热力图
- **功能描述**：展示用户常在哪些时间段复习
- **利用字段**：`ReviewRecord.reviewedAt`
- **实现方式**：按小时聚合统计
- **具体功能**：
  - 日历热力图
  - 学习时段分布图

### 12. 智能提醒时间
- **功能描述**：根据用户学习习惯智能调整提醒时间
- **利用字段**：`ReviewRecord.reviewedAt`, `AppSettings`
- **实现方式**：分析活跃时间段推荐
- **具体功能**：
  - 分析用户活跃时间段
  - 最佳提醒时间推荐
  - 一键应用推荐时间

---

## 🟢 P3 - 低优先级

### 13. 成就系统
- **功能描述**：基于学习数据设计成就徽章
- **利用字段**：`ReviewRecord.reviewedAt`, `Question.reviewCount`, `stageIndex`
- **实现方式**：基于统计数据计算
- **具体功能**：
  - "连续学习7天"成就
  - "复习100题"成就
  - "掌握10个问题"成就
  - 成就徽章展示页

### 14. 复习专注模式
- **功能描述**：减少干扰，提升复习效率
- **利用字段**：`ReviewRecord.responseTimeMs`
- **实现方式**：新增设置项
- **具体功能**：
  - 限制每道题的思考时间（超时提醒）
  - 番茄钟式复习（每10题后休息）

### 15. 临近到期提醒
- **功能描述**：对即将到期的问题提前提醒
- **利用字段**：`Question.dueAt`
- **实现方式**：扩展 ReminderWorker
- **具体功能**：
  - 当天早上提醒当天待复习内容
  - 明日到期问题预览提醒

### 16. 选择性导入
- **功能描述**：选择性导入备份数据
- **利用字段**：全部字段
- **实现方式**：扩展导入逻辑
- **具体功能**：
  - 选择性导入卡组/卡片/问题
  - 导入预览（显示将导入的内容）

---

## 总结：开发顺序建议

| 阶段 | 功能 | 预计改动范围 |
|------|------|-------------|
| **Sprint 1** | 复习统计页面、问题搜索、熟练度标签 | DAO + UseCase + UI |
| **Sprint 2** | 错题本、标签筛选、批量操作 | Repository + UI |
| **Sprint 3** | 复习历史、CSV 导出、响应时间分析 | DAO + Service + UI |
| **Sprint 4** | 热力图、智能提醒、成就系统 | Service + UI |
| **后续** | 专注模式、临近提醒、选择性导入 | 视用户反馈而定 |
